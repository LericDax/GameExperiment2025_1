Procedural Block World

The game now lives entirely inside the Vite workspace under three-demo/. Use one of the workflows below to launch it.

  1. Development: cd three-demo && npm install && npm run dev, then open the URL Vite prints in your terminal (typically http://localhost:5173/).
  2. Production build: npm run build and serve the generated files in three-demo/dist/ with your preferred static server.

This landing page replaces the old CDN-driven build so that the codebase has a single authoritative entry point.
